# Mario Kart simulator 🎮

## Description

### I created this project to improve my JavaScript skills with Node.js, serving as a kind of benchmark. This project is a simple and fun Mario Kart simulator.

## 🕹️ Rules and mechanics

### Players:
#### The computer needs to receive two characters to challenge the race, both are objects
  
### Clues
- The characters will compete on a random track each  of the five rounds
- In each round, a section of the track will be defined, which can be straight, curved, or a combat course.
  - If the defined stretch is straight, the player must roll a 6-sided dice and add the speed attributes. The winner scores a point.
  - If the defined stretch is curve, the player must roll a 6-sided dice and add the maniveurability attributes. The winner scores a point.
  - If the defined stretch is combat, the player must roll a 6-sided dice and add the strength attributes. The winner scores a point.
  - No players can have negative scores (values below zero)


### Winner condition
- At the end, the winner was the one who obtained the highest score
